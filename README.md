# Recaptcha-Website

This projeect tests out the performance of deep learning object detection models in bypassing recaptcha without humans.

Also built a website using HTML, CSS and Flask so that users can upload images and find out objects with specific images. Currently the model only supports three classes.

The model used to detect objects is YOLO-v8. It has been trained again on specific labels of COCO dataset namely traffic lights, motorcycles and Buses.

The new trained model has performed better than the original one and can be viewed in the jupyter notebook.
